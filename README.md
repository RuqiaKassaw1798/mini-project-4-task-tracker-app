# Task Tracker React App

A simple **React Task Tracker** to add, complete, and delete tasks, with a **Stats page** showing total and completed tasks.  

---

## Features
- Add, complete, delete tasks
- Stats page: Total and Completed tasks
- Responsive layout
- Global state via Context API
- Persistent tasks with localStorage

Complete Task: Check the box
